Ayuda y Formatos Admitidos

Para garantizar el análisis correcto, sus archivos deben poseer una estructura básica y nombres de campos lógicos para que el motor inteligente de mapeo de EcoWeigh pueda procesarlos:

1. CSV y Excel (.xlsx, .xls)

Debe incluir encabezados comprensibles en la primera fila (ej. Fecha/Hora y Neto para pesadas; o Fecha y Peso Planta para planta).

2. XML (.xml)

Se busca el elemento de fila repetitivo de forma automática. Sus nodos internos deben llamarse con términos equivalentes:

<row>
  <fecha_hora>2026-05-18 07:15:00</fecha_hora>
  <neto>125</neto>
</row>

3. Informes PDF (.pdf)

El sistema puede extraer tablas de informes PDF de forma nativa utilizando detección de coordenadas espaciales si el archivo posee capa de texto seleccionable (no escaneado plano sin OCR). Buscará de manera inteligente los patrones de fechas y números de pesaje en cada fila visual.

¿No tiene archivos a mano?

Haga clic en el botón "Cargar Simulación Demo" en la parte superior derecha de la pantalla para crear datos de prueba con errores de auditoría de forma automática.
